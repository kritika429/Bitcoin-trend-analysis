# ₿ Bitcoin Market Trend Analysis

## 📌 Overview

Bitcoin Market Trend Analysis is a Data Analytics project developed using Python to analyze over 5 years of historical Bitcoin market data containing 500K+ records. The project focuses on identifying market trends, volatility patterns, price movements, and trading insights through Exploratory Data Analysis (EDA) and interactive visualizations.

The project uses powerful data analysis and visualization libraries such as Pandas, NumPy, Matplotlib, and Seaborn to transform raw cryptocurrency data into meaningful business insights.

---

# 🚀 Features

* Data Cleaning & Preprocessing
* Missing Value Handling
* Exploratory Data Analysis (EDA)
* Bitcoin Price Trend Analysis
* Volatility Analysis
* Moving Average Analysis
